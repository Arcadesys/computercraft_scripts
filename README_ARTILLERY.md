# Artillery Game

A 2-player turn-based artillery game using Pine3D.

## Requirements

- **Pine3D**: This game requires the Pine3D library.
  To install it, run:
  ```
  pastebin run qpJYiYs2
  ```

## How to Play

1. Run the game: `artillery`
2. Player 1 (Blue) starts on the left. Player 2 (Red) starts on the right.
3. On your turn, you can:
   - **A**: Adjust Angle (elevation).
   - **V**: Adjust Velocity (power).
   - **M**: Move your tank (left or right).
   - **F**: Fire!
4. Try to hit the opponent!

## Controls

- **A**: Set Angle
- **V**: Set Velocity
- **M**: Move
- **F**: Fire
- **Q**: Quit

## Models

The game uses custom models defined in `models/tank.lua` and `models/projectile.lua`.
